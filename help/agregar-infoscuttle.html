Crea un bloque donde se configura una búsqueda de enlaces basada en la URL del servidor de scuttle.

La personalización de este bloque se basa en:

  • Título
  • Descripción
  • Mostrar Descripción: se selecciona el lugar donde se va a mostrar la Descripción.
  • URL: se selecciona la URL del servidor de scuttle.
  • Filtrar por: todas las categorías o algunas de las categorías escritas por usted y separadas por coma.
  • Usuario: el nombre del usuario del cual usted desea recuperar los enlaces.
  • Avanzado: filtra por fecha.
  • Ver resultados: cantidad de enlaces a mostrar en el bloque. Resultados como: Seleccione el formato para mostrar los resultados. Página de resultados: muestra una lista de resultados (títulos, descripciones y fechas) y la paginación. Lista de Items: muestra una lista de los títulos y fechas de los eventos. Enlace más...: permite el acceso a los otros resultados. Ordenar por: Cómo ordenar los resultados.
  • Región: se asigna el bloque a alguna región para que se muestre.
